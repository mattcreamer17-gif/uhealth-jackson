# UHealth/Jackson landing page

A single-page, static site built for the UHealth/Jackson booth conversation at eMerge Americas 2026. It carries the fit-analysis pitch body in the prospect's own brand tokens (UM green #005030, UM orange #F47321, warm gray #BEBFB1, din-2014 typography with Barlow and Helvetica Neue fallbacks), opens with the verbatim fit-analysis headline, and routes the prospect to a 30-minute call with Matt Creamer, CRO of Eduba. Eduba branding is confined to the footer and the named case study references, per sprint chrome rules. Pricing and revenue terms have been stripped; KPMG UK is framed as Big Four; NLP Logix is cited with the mandatory "in machine learning since 2011" and "over 150 data scientists" context; the Ethics Engine reference carries its one-sentence bio and hyperlinks to arxiv.org/abs/2510.11742 and github.com/RinDig/AuditEngine.
